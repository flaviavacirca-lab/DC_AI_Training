Non-Copilot AI Tools

Other AI tools consultants may encounter. Understand when they're useful and how to use them safely.

Data Security Rule

Client data must only be used with Microsoft Copilot. External AI tools should only receive public information, generic examples, or fully anonymized scenarios. Never paste client names, financials, proprietary data, or engagement details into non-Copilot tools.

ChatGPT

by OpenAI

Best Use Cases

  • Brainstorming and ideation on generic topics
  • Drafting non-client content (blog posts, internal comms)
  • Learning new concepts or getting explanations
  • Code generation and debugging

Key Strengths

Strong general-purpose reasoning. Good at creative writing, long-form content, and multi-turn conversations. Available via web and mobile apps.

Example Tasks

“Explain the difference between TAM/SAM/SOM with examples” or “Generate 10 icebreaker questions for a strategy workshop.”

Safe to share: Public info, generic questions, anonymized scenarios only

Claude

by Anthropic

Best Use Cases

  • Analyzing long documents (up to 200K tokens)
  • Nuanced writing with precise tone control
  • Structured analysis and synthesis tasks
  • Reviewing and editing existing content

Key Strengths

Excels at long document analysis, careful reasoning, and following detailed instructions. Strong at maintaining consistent tone across long outputs.

Example Tasks

“Review this public SEC filing and summarize the risk factors” or “Rewrite this generic memo in a more analytical consulting tone.”

Safe to share: Public info, generic questions, anonymized scenarios only

Perplexity

AI-Powered Search

Best Use Cases

  • Quick fact-checking with source citations
  • Getting up-to-date market data and news
  • Researching public company information
  • Finding recent industry reports and statistics

Key Strengths

Combines AI with real-time web search. Every answer includes source citations, making it easy to verify claims. Best for research that needs current data.

Example Tasks

“What were the top SaaS M&A deals in 2025?” or “What is the current market size of the US pet food industry?”

Lower risk: Search queries are public by nature, but still avoid client-specific terms

Image Generation Tools

DALL-E, Midjourney, etc.

Best Use Cases

  • Creating concept visuals for internal brainstorming
  • Generating placeholder graphics for draft decks
  • Visualizing abstract concepts or scenarios
  • Creating icons or illustrations for internal use

Key Strengths

Generate custom visuals from text descriptions. Useful for internal ideation; not for client deliverables without review. DALL-E is available inside ChatGPT; Midjourney via Discord.

Example Tasks

“Create a simple icon representing digital transformation” or “Generate a visual metaphor for market disruption.”

Caution: Do not include client names, logos, or proprietary concepts in prompts

When in Doubt, Use Copilot

Microsoft Copilot is the only AI tool approved for client data at Denneen & Company. If your task involves any client-specific information — names, data, strategy, deliverables — use Copilot. External tools are useful for learning, public research, and generic brainstorming only.
