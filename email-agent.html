Building an Email Assistant with Microsoft Copilot Studio

Build a Copilot Studio agent that helps manage your email workflow — learning your style, triaging incoming messages, and drafting replies for review.

What you'll build

An agent called Email Wingmate that will:

  • Learn your writing style from emails you send
  • Analyze incoming emails and decide if they require a response
  • Draft replies when needed
  • Ignore newsletters or informational emails
  • Ask you before drafting optional responses

Safety by Design

The agent never sends emails automatically. It only drafts responses for your review.

System Architecture

Three components work together to power your email assistant.

🧠

1. Copilot Studio (the brain)

Handles reasoning.

  • Classifies incoming emails
  • Decides whether to respond
  • Drafts responses in your style
⚡

2. Power Automate (the event engine)

Handles automation triggers.

  • Detects when an email arrives
  • Detects when you send an email
  • Sends email data to the agent
📂

3. OneDrive / SharePoint folder

Stores learning data.

  • Exports emails you send
  • Agent uses them as style examples

Final Workflow

Incoming Email

Email arrives
↓
Power Automate trigger
↓
Email text sent to Email Wingmate agent
↓
Agent classifies email

Possible outcomes:

  • Draft response
  • Ignore newsletter
  • Ask user if response needed

Sent Email (Learning Loop)

You send an email
↓
Power Automate exports the email
↓
Saved in knowledge folder
↓
Agent uses it as style examples

Part 1 — Create the Email Agent

  1. Open Copilot Studio

    Go to copilotstudio.microsoft.com

  2. Create a New Agent

    Select: Create → New agent

  3. Name the Agent

    Name: Email Wingmate

  4. Set the Description

    Description: Assists with email triage and drafting responses based on the user's writing style.

  5. Click Create

Part 2 — Configure Agent Instructions

Open the agent. Go to Overview → Instructions → Edit. Replace the instructions with the following:

Agent Instructions:

You are Email Wingmate, an assistant that helps triage and draft responses to incoming emails.

CLASSIFICATION RULES
Classify each email into exactly one category:
CATEGORY_A_NEEDS_RESPONSE
CATEGORY_B_CONTENT_SHARING
CATEGORY_C_OPTIONAL_RESPONSE

ACTION RULES
If CATEGORY_A_NEEDS_RESPONSE
Draft a reply in the user's writing style.
If CATEGORY_B_CONTENT_SHARING
Ignore the email.
If CATEGORY_C_OPTIONAL_RESPONSE
Ask the user whether they want to respond.

SAFETY
You must never send emails automatically.
Only generate draft responses or categorize messages.

STYLE
Write concise, professional responses that match the user's typical writing tone based on prior emails.

Click Save after pasting.

Part 3 — Add Agent Tools

Open Agent → Tools and add the following tools.

Tool 1: Assign an Outlook Category

Purpose: Used to mark newsletters or ignored emails.

Tool 2: Draft an Email Message

Purpose: Creates email drafts.

Important

Do NOT add the Send email tool. The agent should only draft — never send automatically.

Part 4 — Build the Email Triage Topic

Go to Topics → New Topic. Name it Handle incoming email.

  1. Add a Generative Answers Node

    Add node: Advanced → Generative Answers

    Instruction:

    Analyze the email content and classify it into exactly one category.
Return the category name on the first line.
CATEGORY_A_NEEDS_RESPONSE
CATEGORY_B_CONTENT_SHARING
CATEGORY_C_OPTIONAL_RESPONSE
Then briefly explain why.

    Save response as variable: TopicClassification

  2. Add a Condition Node with Three Branches

Condition: TopicClassification contains CATEGORY_A_NEEDS_RESPONSE

Action: Call tool → Draft an email message

Instruction:

Draft a concise reply in the user's writing style.
Answer questions from the email.
Do not send the email.

Condition: TopicClassification contains CATEGORY_B_CONTENT_SHARING

Action: Call tool → Assign Outlook category

Category → Newsletter

Condition: TopicClassification contains CATEGORY_C_OPTIONAL_RESPONSE

Action: Ask: "Would you like to respond to this email?"

  • Yes → Draft email
  • No → Assign category "No Response Needed"

Part 5 — Publish the Agent

  1. Open the Agent

    Navigate to your Email Wingmate agent in Copilot Studio.

  2. Click Publish

    Publishing is required for Power Automate to call the agent.

Part 6 — Create Email Learning Flow

This flow exports your sent emails so the agent can learn your writing style.

  1. Open Power Automate

    Go to make.powerautomate.com

  2. Create an Automated Cloud Flow

    Name: Export Sent Emails for Copilot Learning

  3. Set the Trigger

    Trigger: When a new email arrives (V3)
    Folder: Sent Items

  4. Add Action: Export email (V2)

    Field: Message Id → Message Id

  5. Add Action: OneDrive → Create File

    Folder: Email Style Samples / Sent Emails
    File name: timestamp_subject.eml
    File content: Body from Export Email

Part 7 — Create Incoming Email Flow

This flow sends incoming emails to your agent for triage.

  1. Create a Second Flow

    Name: Process Incoming Email

  2. Set the Trigger

    Trigger: When a new email arrives (V3)
    Folder: Inbox
    Remove filters.

  3. Add Action: Compose

    Input:

    RUN_TOPIC: Handle incoming email
New email received.
From: [From Address]
Subject: [Subject]
Body
[Body Preview]
  4. Add Action: Execute Agent

    Agent: Email Wingmate
    Body: Outputs from Compose

Part 8 — Test the System

Send a test email to verify each classification path works correctly.

Draft Response

Send an email requiring a reply. Check your Outlook Drafts for the agent's response.

Newsletter

Forward a newsletter. Verify it gets categorized as Newsletter in Outlook.

Optional

Send an ambiguous email. The agent should ask you whether to respond.

Safety Design

This system is designed to keep you in control at every step.

Built-in Safeguards

  • No automatic replies — all responses are drafts for your review
  • No unauthorized sending — the Send email tool is never added
  • No sensitive actions without review — all responses remain human-reviewed

Future Improvements

Once your agent is running, consider these enhancements:

  • Newsletter summaries instead of simply ignoring
  • VIP sender rules for priority contacts
  • Response confidence thresholds
  • Weekly ignored email digest
  • Improved style profiling over time

Human Review Required

Email agents generate draft responses automatically. Before sending any AI-drafted email:

  • Verify tone is appropriate for the client relationship
  • Confirm facts and numbers are correct
  • Check that no sources were hallucinated
  • Ensure the classification (Needs Response / Content Sharing / Optional) is accurate

AI-drafted emails should always be reviewed before sending.
